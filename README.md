## 🛍️ Print-on-Demand Integration Plan

### Recommended Platform: **Printful**
- Free to set up, no inventory costs
- Integrates with standalone stores via API
- High-quality DTG (Direct-to-Garment) printing
- Embroidery available for hats

### Setup Steps:
1. Create Printful account
2. Upload designs (300 DPI PNGs with transparency)
3. Set product variants (sizes, colors)
4. Connect to e-commerce platform
5. Set pricing (cost + margin)
6. Configure shipping zones

### Products to Launch:
- Premium ringspun cotton tees ($15 cost → $65 retail)
- Linen-cotton resort shirts ($28 cost → $120 retail)
- Unstructured cotton caps ($8 cost → $42 retail)
- All-over-print button-ups ($32 cost → $120 retail)
